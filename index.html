Сайты-библиотеки для Вас

Предлогаю Вашему вниманию список сайтов, которые помогут не только начинающим, но и бывалым веб-программистам :

  • HTMLBook - на данном сайте есть самоучители по HTML5/CSS3, а так же справочники по ним, есть курсы о том, как сверстать сайт и многое другое
  • W3Schools - здесь вы сможете найти много полезного во всех сферах веб-программирования, уроки/справочники по HTML, CSS, JavaScript (так же фреймворки), PHP
  • CSS-Tricks - на этом сайте вы сожете так же найти много интересного/полезного для себя
  • CSS4-Selectors - здесь же вы найдёте все селекторы которые используются в CSS1-4, а так же совместимость их с браузерами

Сайты-помощники для вёрстки

Сайты с которыми Вам будет необходимо "сотрудничать", для того, чтобы вы, к примеру, могли настроить правильно шрифты и т.д.:

  • Transfonter - один из генераторов шрифтов, с ним очень просто работать, даёт стабильный результат
  • Google Fonts - один из первых сайтов на который бегут все программисты, если конечно нет лучшей альтернативы, для того чтобы скачать/установить шрифты

Программы

Программы которыми пользуюсь/пользовался я, они как по мне, одни из лучших, они относительно не большие и лёгкие в усвоении, а так же в них присутствует огромное количество функционала :

  • PHP Storm - среда разработки, в ней тонны функционала, подходит для больших проэктов
  • ST3 - редактор кода, который Вы влюбитесь с первого взгляда, он прост и так же имеет большое количество функционала, подходит для маленьких и больших проэктов
  • Brackets - так же отличный редактор кода, хорош для новичков, хороший помощник в написании кода, для маленьких и средних проэктов, в больших замечены значительные провисания
  • Smart Git - программа которая обрежет Ваше время для работы с Git-ом
  • Ruby - динамический язык программирования с открытым исходным кодом с упором на простоту и продуктивность, но для нас он требуется для работы с SASS
  • Compass - компилятор SCSS файлов, он переводит синтаксис SCC в обычный CSS
  • SASS - Sass (Syntactically Awesome StyleSheets), препроцессор, имеет улучшенный синтаксис CSS, код становится более читабельней и легче в написании
